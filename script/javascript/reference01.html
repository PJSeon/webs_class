변수란?

말 그대로 변하는 수.

변수는 변하는 데이터(값)를 저장할 수 있는 메모리 공간이다. 데이터를 담을 수 있는 그릇으로, 변수에는 오직 한 개만 저장된다.

새로운 데이터가 들어오면 기존에 있던 데이터는 메모리 공간에서 지워진다.

변수에 저장할 수 있는 데이터의 종류는 문자형(String), 숫자형(Number), 논리형(Boolean) 그리고 빈(Null) 데이터가 있다.

변수 선언

변수를 선언 할 때는 var나 let 키워드를 변수명 앞에 붙인다. 변수명에는 한글을 사용 할 수 없으며, 영문과 숫자 그리고 일부 특수 문자(_, $)만 포함할 수 있다.


[val 변수명; 또는 val 변수명=값;]
[let 변수명; 또는 let 변수명=값;]


변수명은 의미에 맞게 사용하는 것이 좋다. 가령, 수학 시험 점수를 저장하는 변수라면 'mathNum'이라고 이름을 지으면 좋다.

이때 변수명은 단어와 단어의 조합으로 지었고, 두번째 단어의 첫 글자는 대문자로 표기했다. 이는 낙타의 등과 같다고 해서 낙타(Camel) 표기법이라고 한다.

변수에는 하나의 값만 넣을 수 있다. 만약 변수 box에 값을 두 번 넣으면, 기존에 있던 데이터는 삭제 되고 새로운 데이터가 저장 된다. ( 02. 변수 참고(데이터 저장+변경 )

변수에 저장할 수 있는 자료형

문자형

문자형(String) 데이터는 [val 변수="사용할 문자나 숫자";] 처럼, 큰따옴표와 작은따옴표로 감싸고 있어야 한다. 또한 문자형 데이터에 HTML 태그를 포함하여 출력하면 태그로 인식한다.

val s="jabascript";

val num="100";

var tag=" String";


ex)
var x = 100; //변수 x에 숫자 100을 저장
var y = 200; //변수 y에 숫자 200을 저장
var z = "javascript"; //변수 z에 문자 javascript 저장, 문자 입력시에는 " "

document.write("*** 01. 변수 ***");
document.write(x, "br");
document.write(y, "br");
document.write(z, "br");


숫자형

숫자형은(Number) 데이터는 단어 의미 그대로 숫자를 의미한다. 만약 "100"과 같이 큰따옴표가 숫자를 감싸고 있다면 숫자가 아닌 문자형 데이터이다. 이 경우에는 number("100")를 이용하면 문자형 데이터를 숮자형 데이터로 바꿀 수 있다.


ex)

val s=100;

val t=Number("500"); //"500" -> 500


논리형

논리형(Boolean) 데이터는 true(참) 또는 false(거짓)가 있다. 이 데이터는 주로 2개의 데이터를 비교할 때 나오는 결과이다.

예를 들어 '100보다 10이 크다'는 잘못된 비교이므로 false라는 결과를 반환하게 된다.


ex)

var s=true;

var t=10>=100; //false

var k=Boolean("hello"); //true


null & undefined 데이터

undefined는 변수 s에 값이 등록되기 전의 기본값이고, null은 변수에 저장된 값이 null인 경우를 가리킨다. null은 변수에 저장된 데이터를 비우고자 할 때 사용하는 값이다.


typeof

typeof는 지정한 데이터 또는 변수에 저장된 자료형을 알고 싶을 때 사용한다. 기본형은 다음과 같다.

typeof 변수 또는 데이터;
