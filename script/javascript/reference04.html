식별자/키워드

식별자

이름을 붙일 때 사용하는 단어로, 프로그래머가 선언하는 단어이다.

변수나 함수의 이름을 작성할 때 쓰인다.

고유하게 식별 될 수 있도록 프로그램의 실체를 제공한 이름으로, 변수, 함수, 클래스, 레이블 및 기타 다양한 사용자 정의 객체의 이름을 가진다.


식별자에는 아래와 같은 규칙이 있다.


식별자 규칙


    1. 키워드를 사용하면 안 된다.
    2. 숫자로 시작하면 안 된다.
    3. 특수문자는 _, $만을 허용한다.
    4. 공백 문자를 포함할 수 없다.

키워드

자바스크립트 안에 이미 존재하는 예약된 단어이다. (예약어라고도 한다)

즉, 이 키워드는 식별자의 이름을 지정하는 데 사용 할 수 없다. 그리고 각 키워드는 별개의 의미를 가진다.


28가지 키워드


    break, if, else, instanceof, true, false, switch, case, default, try, catch, finally, null, typeof, continue, for, return, var, new, function, void, delete, this, while, do, in, throw, with
