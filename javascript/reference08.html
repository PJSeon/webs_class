예제 - for문을 이용해 1부터 10까지 i가 2의 배수일 경우에만 continue 실행하여 홀수만 출력 
<script>
    for(var i = 1; i <=10; i++) {
        if(i % 2 == 0 ) continue;       //continue 문은 반복문 이후의 코드 실행을 건너뛰고 다시 반복문의 증감식을 수행합니다.
        document.write(i, "<br>");
    }
    document.write("===The End===");
</script>
