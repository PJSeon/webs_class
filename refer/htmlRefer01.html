HTML이란?

HTML은 HyperText Markup Language의 약자로 웹페이지를 구성하는 마크업 언어입니다

웹 페이지는 HTML 문서라고도 불리며, HTML 태그들로 구성됩니다.

HTML은 태그와 속성, 내용과 요소로 구성됩니다.

HTML의 기초 문법

HTML은 기본적으로 요소들로 구성 되어 있습니다. 이러한 요소들은 시작 태그와 종료 태그의 쌍으로 구성 되어 있으며 태그는 < > 로 둘러 싸인 키워드입니다.

아래는 태그의 가장 기본적인 형태입니다.

<head>
<title> 제목 </title>
</head>

HTML의 주석표시

HTML 역시 주석을 사용할 수 있습니다. 주석은 코드를 이해하는데 도움을 주며, 웹 페이지에는 표시되지 않습니다.

아래는 주석을 사용하는 방법입니다.

 <!--주석-->
