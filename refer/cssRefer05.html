id

id는 한페이지의 하나의 정의로 단 하나의 태그(id="")만 사용할 수 있습니다.

로고, 상단메뉴, 하단정보 같은 스타일을 정의할 때 사용합니다.

id 선언자는 #을 사용합니다.

class

class는 한페이지에 반복적으로 사용되는 스타일을 정의할 때 사용합니다.

반복적으로 사용되는 스타일에는 class를 정의하고, 그 내부에 세부 스타일을 정의할 때 id를 사용해주면 효과적입니다.

class 선언자는 .을 사용합니다.
